# Sufe CLI - 成绩查询技能

**⚠️ 重要提示：在执行本技能中的任何命令前，请务必先参考并触发 `sufe-base` 技能，以确保 CLI 环境已正确就绪并获取了有效的用户认证 (Cookie)。**

本技能介绍如何使用 `sufe score` 命令组来查询上海财经大学教务系统中的历史课程成绩与学期绩点汇总。

## 1. 核心概念

- **成绩汇总**：按学期统计的汇总数据，包含门数、平均成绩、总学分、平均绩点。
- **详细成绩**：每门课程的完整成绩记录，包含课程代码、课程名称、课程类别、学分、总评成绩、最终成绩、绩点等字段。
- **TSV 格式**：所有输出均以制表符（`\t`）分隔，便于直接导入 Excel、Numbers 或进行管道处理。

## 2. 常见指令及其说明

| 指令 | 说明 |
|---|---|
| `sufe score summary` | **学期成绩汇总**：获取各学期的门数、平均成绩、总学分、平均绩点汇总（TSV 格式）。 |
| `sufe score list` | **全部课程成绩**：获取所有历史课程的详细成绩列表（TSV 格式）。 |
| `sufe score list --semester "2025-2026 1"` | **指定学期成绩**：按学年学期筛选，仅输出该学期的课程成绩。 |

## 3. 示例工作流

当用户要求查看成绩时，推荐的工作流如下：

1. **查看学期汇总**：
   了解各学期的整体学业表现（平均成绩、绩点等）：
   ```bash
   sufe score summary
   ```
   输出示例（TSV）：
   ```
   学年度	学期	门数	平均成绩	总学分	平均绩点
   2023-2024	1	9	86.2	25	3.54
   2023-2024	2	12	90.24	29	3.77
   2024-2025	1	12	90.57	28.5	3.88
   ```

2. **查看全部课程成绩**：
   获取完整的课程成绩明细：
   ```bash
   sufe score list
   ```
   输出示例（TSV）：
   ```
   学年学期	课程代码	课程序号	课程名称	课程类别	学分	总评成绩	最终	绩点
   2025-2026 1	102041	1135	财政学	选修课	2	97	97	4
   2025-2026 1	103432	0821	金融中国-通识篇	通识模块四（经济分析与数学思维）限选课	2	80	80	3
   2024-2025 2	102876	1078	深度学习	选修课	3	99	99	4
   ```

3. **查看特定学期成绩**：
   若只想查看某一学期的成绩，使用 `--semester` 参数：
   ```bash
   sufe score list --semester "2025-2026 1"
   ```

4. **处理异常**：
   如果命令提示登录状态已过期，请查阅 `sufe-base` 并引导用户运行 `sufe auth` 重新获取门户状态。
